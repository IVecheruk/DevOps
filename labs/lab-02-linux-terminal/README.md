## Цель работы

Приобрести опыт работы с командной строкой Linux, изучить основные команды терминала и закрепить практические навыки администрирования: просмотр даты и времени, получение информации о системе, управление пользователями и анализ процессов.

## Теоретические сведения

Терминал Linux предоставляет текстовый интерфейс для взаимодействия с операционной системой через командную оболочку. В этой среде пользователь может выполнять системные утилиты, получать сведения о состоянии компьютера, работать с учётными записями и анализировать активные процессы.

Справка по утилитам обычно доступна через `man` или через ключ `--help`, который позволяет быстро посмотреть параметры запуска. Для повседневного администрирования важны команды работы со временем и системой, а также команды `hostname`, `uptime`, `ps`, `whoami`, `users`, `less` и другие.
