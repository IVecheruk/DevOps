# Source Materials

Каталог хранит исходные материалы, из которых были подготовлены Markdown-отчёты:

- `docx/` — исходные версии лабораторных работ в формате Word;
- `legacy-images/` — старые отдельные папки со скриншотами.

Эти файлы сохранены как архивные материалы и могут использоваться для сверки с итоговыми отчётами в каталоге [`../labs`](../labs).
